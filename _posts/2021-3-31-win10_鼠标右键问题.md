# windows10更新后鼠标右键一直转圈圈，一段时间后直接刷新整个屏幕 

### 解决步骤  
win + R 输入  regedit打开注册表 
依次点击 
HKEY_CURRENT_USER 
SOFTWARE
Microsoft 
Windows 
CurrentVesion 
Policies 
Explorer 
双击右侧的NoDriverType，清空数值数据 
保存重启解决问题
